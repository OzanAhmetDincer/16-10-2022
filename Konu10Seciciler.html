Js Seçiciler

Javascript ile DOM(Document Object Model) üzerinden elementler üzerinde hızlıca maniplasyon yapabilir. Bunun için document yani sayfa üzerindeki istediğimiz elementi seçmemiz yeterlidir.

Javascript de sayfadaki elementlerin herhangi bir özelliği ile elementi seçebiliriz. Id, name, class veya alt-üst ilişkili olduğu bir element üzerinden.
